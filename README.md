## Introduce

1. 这是一个vuepress主题，它基于vuepress-theme-reco主题做了改造；

## Quick start

**package.json** 内添加

```json
"vuepress-theme-tz": "^1.0.0",
```

## 扩展内容

### 锚点

增加10种主页锚点样式

![anchor-doc](.\example\.vuepress\public\picture\anchor-doc.png)

**使用方式**

```js
// config.js
...
themeConfig: {
    anchorType: 7 // 这里的数字对应图上的数字
}
...
```
